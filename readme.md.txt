1) Open a terminal in the algoGames folder and type `npm install` and press Enter.

2)Change directory by typing `cd server` and type npm install and enter.

3)Create database named algogamesdb in your mysql server.

4)load tables and data if you want using the Dump file in the db_dump folder. Or else when you are playing, the tables will be created automatically.

5)Go to server folder and change your db configurations such as username, password according to your mysql server configurations.

6)Open a terminal in the algoGames folder, type `npm start` and enter.

7)Change directory by typing `cd server`, type `npm start` and enter.

8)browser will be opened in the localhost port and enjoy the games.
